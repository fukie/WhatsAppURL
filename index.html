WhatsApp URL Generator

Say hello to new connections effortlessly on WhatsApp!

Start With 3 Simple Steps!

  1.  Enter your contact's phone number and country below.
  2.  Select 'Generate' or 'Generate & Open WhatsApp'
  3.  Click on the generated URL.

Details

Some details about this tool:

  • Numbers are NOT stored by this tool.
  • This tool doesn't verify if the number is in the valid format or if it is working.
  • Including the country code prefix in the number is optional.
  • Spaces and dashes in the number are automatically handled.
  • The country code would default to your IP address' country (usually within ~5 seconds).
  • The country code dropdown is searchable.
  • Click on the generated URL which should trigger WhatsApp (if it is installed) to open on your mobile deviecs or computers.
