Generate WhatsApp Send URL

Instructions:

1. Select the correct country/country calling code.
 - Tip: You could type the partial country name or calling code to filter the list and select.
 - It would default (within a few seconds) to the country of your IP address, unless detection is blocked by your browser (e.g., uBlock or Brave Shields).
2. Enter a valid phone number.
 - Spaces, dashes or mobile prefix are accepted and would be handled accordingly.
3. Click 'Go' and a URL would be generated and automatically opened in a new tab.
 - If it doesn't, select/copy the generated URL into your preferred Internet browser.

Numbers are NOT stored by this tool.
